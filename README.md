CS To-Do List | Developer Edition 💻

A simple, clean, and classic To-Do List app — made just for fun, inspired by how traditional coders used to start their journey.

It’s one of those nostalgic projects every developer builds at the beginning — but with a modern CS-student twist: clean UI, smooth interactions, and a bit of personality.

⚙️ Features

✅ Add, edit, and delete tasks

✅ Mark tasks as complete

✅ Filter by All / Active / Completed

✅ Auto-save using LocalStorage

✅ Dark / Light theme toggle 🌙

✅ Smooth modern UI with gradient touches ✨

🎨 Tech Stack

HTML

CSS (Glassmorphism + Gradients)

JavaScript (LocalStorage, Filters, Theme Toggle)

💬 About

This project was created just for fun — a little throwback to the kind of projects that helped every coder learn the basics.
Simple logic, clean code, and a touch of creativity — the coder’s way. 🚀
